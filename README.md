# Pairing-Based Multi-Omics Fusion for Cancer Subtype Discovery

**Original project period:** September 2024–April 2025  
**Public archive consolidated:** July 2026

## Overview

This project explored cancer-subtype discovery by integrating four TCGA omics modalities:

- mRNA expression;
- miRNA expression;
- copy-number alteration;
- DNA methylation.

The research workflow used a multi-input autoencoder and pairwise fusion design, then compared similarity-matrix fusion with feature-concatenation strategies. Downstream analysis included Gaussian-mixture and consensus clustering, survival analysis, and pathway-level interpretation.

## Public scope

This repository is a curated public summary rather than a complete release of the original collaborative research assets. Raw data, some preprocessing outputs, implementation details, and joint project materials remain private because they include large datasets and collaboratively developed assets that were not cleared for redistribution.

## Research components

1. Modality-specific preprocessing and representation learning.
2. Multi-input autoencoder construction.
3. Pairwise and aggregate fusion across four omics modalities.
4. Comparison of similarity-matrix fusion and feature concatenation.
5. Clustering with GMM and consensus-based methods.
6. Downstream survival and pathway analysis.
